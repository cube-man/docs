# git 的使用

## git 介绍

- git 是一种 vcs（版本控制工具）

## 版本控制

### 版本更新规则

- 版本一般包含 3 部分，如 release/V1.0.0_xxxx20220711
- 1.0.1 bug 的修复
- 1.1.0 添加新的 api
- 2.0.0 结构的变化

## 一些指令

1. git init：初始化当前目录为仓库，初始化后会将当前仓库设置为 master
2. git add "文件位置或文件名"：将文件添加到缓存区
3. git commit -m "描述信息"：提交到本地仓库
4. git commit --amend：重写上次提交的信息
5. git push：推送的远程仓库
6. git fetch:拉取分支信息
